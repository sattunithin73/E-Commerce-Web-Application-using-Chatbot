# 🛒 AI-Powered E-Commerce Web Application

A full-stack e-commerce web application built using Django with an integrated chatbot to assist users in searching and exploring products.

---

## 🚀 Features

- 🔐 User Authentication (Login, Signup, Logout)
- 🛍️ Product Listing with Dynamic Rendering
- 🔍 Product Search Functionality
- 🤖 AI Chatbot for Product Assistance
- 📄 Separate Pages (Home, About, Contact, Orders)
- 📱 Responsive UI Design
- 🧭 Smooth Navigation and Clean UI

---

## 🧠 Chatbot Capabilities

- Understands user queries like:
  - "cheap products"
  - "formal shirts"
  - "jeans"
- Filters products based on:
  - Name
  - Price conditions
- Provides instant responses
